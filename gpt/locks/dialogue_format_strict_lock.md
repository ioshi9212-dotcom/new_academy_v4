# Lock: строгий формат реплик

Этот файл имеет высокий приоритет для любого сценового ответа.

## Главный принцип

Шапка сцены недостаточна.

Если в сцене есть реплики, каждая реплика должна быть оформлена строго так:

```txt
**Имя или видимый дескриптор** — Реплика. (*короткая ремарка*)
```

Жирное имя обязательно.

Длинное тире обязательно.

Ремарка необязательна, но если она есть — только короткая, в круглых скобках и курсивом.

## Ремарки

Ремарка в реплике — это не действие на пол-абзаца.

Ремарка должна быть короткой:

```txt
(*тихо*)
(*сухо*)
(*смеётся*)
(*усмехается*)
(*смотрит в сторону*)
(*после паузы*)
(*не поднимая голоса*)
```

Ремарка должна быть именно в формате:

```txt
(*текст ремарки*)
```

Не писать ремарки обычным текстом:

```txt
(тихо)
(смеётся)
(смотрит в сторону)
```

Не писать длинные ремарки:

```txt
(*она долго смотрит на него, потом вспоминает прошлую сцену и решает не отвечать*)
```

Такое нужно выносить отдельным описанием курсивом.

## Описания

Описание действия, атмосферы, движения, паузы или окружения писать отдельной строкой курсивом:

```txt
*Ливия опирается плечом о стену и быстро смотрит на Акиру.*
```

Описание не должно заменять формат реплики.

## Правильно

```txt
*Ливия опирается плечом о стену и быстро смотрит на Акиру.*

**Ливия** — Ты опять смотришь так, будто уже нашла три выхода. (*усмехается*)

**Хару** — Это талант или угроза? (*смотрит с интересом*)

**Акира** — Привычка.
```

Если имя персонажа Акира ещё не знает, использовать видимый дескриптор:

```txt
**Рыжий старшекурсник** — Новенькие? (*останавливается у края площадки*)
```

## Неправильно

```txt
Ливия — Ты опять смотришь так...

Ливия: Ты опять смотришь так...

- Ты опять смотришь так, — сказала Ливия.

Ливия усмехнулась: Ты опять смотришь так...

**Ливия** — Ты опять смотришь так. (усмехается)

**Ливия** — Ты опять смотришь так. (*усмехается, потом подходит ближе, вспоминает вчерашний разговор и решает сменить тему*)
```

## Описания отдельно

Действия и атмосфера не должны смешиваться с репликами.

Правильно:

```txt
*Ливия опирается плечом о стену и быстро смотрит на Акиру.*

**Ливия** — Не делай вид, что ты этого не заметила. (*тихо*)
```

Неправильно:

```txt
Ливия опирается о стену и говорит — Не делай вид...
```

## Проверка перед отправкой

Перед отправкой сцены ИИ обязан проверить:

- каждая реплика начинается с `**`;
- имя/дескриптор закрыт `**`;
- после имени стоит длинное тире `—`;
- реплика не оформлена обычным текстом;
- нет формата `Имя:`;
- нет формата `Имя —` без жирного имени;
- нет книжного формата с кавычками вместо сценового диалога;
- все ремарки в репликах оформлены как `(*коротко*)`;
- нет ремарок вида `(тихо)` без курсивных звёздочек;
- нет длинных действий внутри скобок;
- описания действий и атмосферы вынесены отдельными курсивными строками.

Если хотя бы одна реплика оформлена не так — сцену нужно переписать до отправки.

Ход с неправильным форматом реплик считается технически неправильным, даже если шапка оформлена верно.
